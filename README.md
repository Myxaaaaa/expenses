# 💰 Telegram бот для учета расходов в группах

Бот для учета расходов в Telegram группах. Позволяет легко добавлять расходы и просматривать статистику.

## 🚀 Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Получите токен бота у [@BotFather](https://t.me/BotFather) в Telegram

3. Установите токен бота одним из способов:
   - Через переменную окружения:
     ```bash
     # Windows PowerShell
     $env:BOT_TOKEN="ваш_токен_бота"
     
     # Windows CMD
     set BOT_TOKEN=ваш_токен_бота
     
     # Linux/Mac
     export BOT_TOKEN="ваш_токен_бота"
     ```
   - Или отредактируйте файл `bot.py` и замените строку:
     ```python
     BOT_TOKEN = os.getenv("BOT_TOKEN", "ваш_токен_бота")
     ```

4. Добавьте бота в группу и дайте ему права на чтение сообщений

5. (Опционально) Настройте прокси для обхода блокировки Telegram API:
   ```bash
   # Windows PowerShell
   $env:PROXY_URL="socks5://proxy.example.com:1080"
   $env:PROXY_USERNAME="username"  # Опционально
   $env:PROXY_PASSWORD="password"  # Опционально
   
   # Windows CMD
   set PROXY_URL=socks5://proxy.example.com:1080
   set PROXY_USERNAME=username
   set PROXY_PASSWORD=password
   
   # Linux/Mac
   export PROXY_URL="socks5://proxy.example.com:1080"
   export PROXY_USERNAME="username"
   export PROXY_PASSWORD="password"
   ```
   
   Поддерживаемые форматы прокси:
   - SOCKS5: `socks5://proxy.example.com:1080`
   - HTTP: `http://proxy.example.com:8080`
   - С авторизацией: `socks5://username:password@proxy.example.com:1080`

6. Запустите бота:
```bash
python bot.py
```

## 📝 Использование

### Добавление расходов

Чтобы бот записал расход:
1. Ответьте на сообщение/фото, к которому относится расход
2. Укажите сумму и описание
3. Обязательно упомяните бота (`@имябота`)

Пример сообщения: `@bot 1500 продукты`

Бот сохранит расход в сомах и пришлёт подтверждение со ссылкой на исходное сообщение.

### Команды

- `/expenses` – расходы за сегодня
- `/expenses_week` – расходы за последние 7 дней
- `/expenses_month` – расходы за последние 30 дней
- `/expenses_period 01.01.2024 07.01.2024` – расходы за произвольный период
- `/delete <ID>` – удалить расход по ID (администраторы и шефы могут удалять чужие расходы)
- `/setrole <роль>` – назначить роль (только админы чата, команда выполняется в ответ на сообщение участника)
- `/roles` – список ролей в группе
- `/test` – проверка, что бот в сети

## 📊 Особенности

- ✅ Расходы учитываются только по ответам с упоминанием бота — меньше случайных записей
- ✅ Все суммы хранятся в сомах
- ✅ Ссылки на исходные сообщения для быстрой проверки
- ✅ Фильтры по дням, неделям, месяцам и произвольным периодам
- ✅ Назначение ролей (оператор, администратор, шеф) только администраторами
- ✅ Администраторы и шефы могут удалять любые расходы
- ✅ База данных SQLite (файл `expenses.db`)
- ✅ Поддержка прокси для обхода блокировки Telegram API

## 🔧 Технические детали

- Python 3.7+
- python-telegram-bot 20.7
- SQLite для хранения данных
