# 🔐 Настройка прокси для обхода блокировки Telegram API

Если Telegram API заблокирован в вашей стране или сети, вы можете настроить прокси-сервер для работы бота.

## 📋 Поддерживаемые типы прокси

- **SOCKS5** (рекомендуется): `socks5://proxy.example.com:1080`
- **HTTP/HTTPS**: `http://proxy.example.com:8080`

## ⚙️ Настройка через переменные окружения

### Windows PowerShell

```powershell
# Базовый вариант (без авторизации)
$env:PROXY_URL="socks5://proxy.example.com:1080"

# С авторизацией (вариант 1 - в URL)
$env:PROXY_URL="socks5://username:password@proxy.example.com:1080"

# С авторизацией (вариант 2 - отдельные переменные)
$env:PROXY_URL="socks5://proxy.example.com:1080"
$env:PROXY_USERNAME="username"
$env:PROXY_PASSWORD="password"
```

### Windows CMD

```cmd
set PROXY_URL=socks5://proxy.example.com:1080
set PROXY_USERNAME=username
set PROXY_PASSWORD=password
```

### Linux/Mac

```bash
export PROXY_URL="socks5://proxy.example.com:1080"
export PROXY_USERNAME="username"
export PROXY_PASSWORD="password"
```

## 🔧 Примеры конфигурации

### Пример 1: SOCKS5 прокси без авторизации
```bash
PROXY_URL=socks5://127.0.0.1:1080
```

### Пример 2: HTTP прокси с авторизацией
```bash
PROXY_URL=http://proxy.example.com:8080
PROXY_USERNAME=myuser
PROXY_PASSWORD=mypass
```

### Пример 3: SOCKS5 с авторизацией в URL
```bash
PROXY_URL=socks5://user:pass@proxy.example.com:1080
```

## 🚀 Запуск бота с прокси

После настройки переменных окружения просто запустите бота как обычно:

```bash
python bot.py
```

Бот автоматически обнаружит настройки прокси и будет использовать их для подключения к Telegram API.

## ✅ Проверка работы

При запуске бота вы увидите сообщение:
```
✓ Прокси настроен
```

Если прокси не настроен или не указан, бот будет работать напрямую.

## ⚠️ Важные замечания

1. **Безопасность**: Не публикуйте свои учетные данные прокси публично
2. **Производительность**: Использование прокси может немного замедлить работу бота
3. **Надежность**: Убедитесь, что прокси-сервер стабильно работает
4. **Ошибки**: Если прокси не работает, бот попытается подключиться напрямую (если возможно)

## 🔍 Решение проблем

### Проблема: Бот не может подключиться через прокси

**Решение:**
1. Проверьте правильность адреса и порта прокси
2. Убедитесь, что прокси-сервер доступен
3. Проверьте правильность учетных данных (если требуется авторизация)
4. Проверьте логи бота для детальной информации об ошибке

### Проблема: Прокси настроен, но бот все равно не работает

**Решение:**
1. Убедитесь, что прокси поддерживает SOCKS5 или HTTP
2. Проверьте, что прокси не блокирует подключения к `api.telegram.org`
3. Попробуйте другой прокси-сервер

## 📚 Дополнительная информация

Для получения прокси-сервера вы можете:
- Использовать VPN с SOCKS5 поддержкой
- Настроить собственный прокси-сервер
- Использовать публичные прокси (не рекомендуется для безопасности)
